An overview of all the computing tools I currently employ for projects
and for play as well. My philosophy regarding computers or technology in
general is not having the most powerfull tool in the shed - I find more
joy in re-speccing old gear for modern usage.

Some aspects I don't mind having the best of the best, like guitars or where
performance really matters (like cars or most electronics) - the bliss of squeezing
every bit of power you can from a more weathered machine is absolutely fantastic.

This my daily driver, I mainly use to program, techie stuff or browse the

doing stuff but I feel like it's more of a Windows 11 problem than a computer
problem because if I reinstall Windows 7 or Arch on this, performance would not
be an issue.

I use Windows Terminal, VSCode, and Office Suite 97 mostly on this. I run
Google Chrome as my main browser as well so I don't have to worry about
syncing my stuff and I faily use Google tools alot. I am now using the Scoop
package manager mostly. I also use this for work sometimes but I might but
a laptop for that as I tend to reset my computer and I don't want to have to
call my IT guy every month or so LOL.

As well, I have a couple of mechanical keyboard and mice that I switch around
whenever I feel like but I only use mechanical keyboard and gaming mices. I'm
a firm believer of tactile responses and muscle memory so I tend to use the
same tools over and over again.
